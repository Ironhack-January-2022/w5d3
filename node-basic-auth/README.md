### Hashing

```
<any digital input> -> '$2b$10$b/0PX8UUYwzY20sxfJYzu.IAWTqCHcbh1EHJdr0keknqk1fKhOVHC'
```

- deterministic 
- fixed length 
- small change in input results in a completely different hash
- **cannot be decrypted back to it's original input**
